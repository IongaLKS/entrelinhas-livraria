# Sistema de Estoque

Sistema desenvolvido em PHP seguindo o padrão MVC.

## Funcionalidades

- Cadastro de produtos
- Edição de produtos
- Exclusão de produtos
- Busca por nome
- Login de usuários

## Tecnologias

- PHP
- MVC
- PDO
- MySQL
- HTML
- CSS

## 🚀 Como executar

1. Clone este repositório:

bash
git clone https://github.com/La-vielina/entrelinhas-livraria.git


2. Coloque a pasta do projeto em htdocs (XAMPP).

3. Inicie o Apache e o MySQL.

4. Configure o banco de dados.

5. Acesse no navegador:


http://localhost/entrelinhas
